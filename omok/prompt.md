# 웹 오목 게임 프롬프트 v1.01

## v1.00 에서 변경된 점
- 흑돌 전용 삼삼(33) 금지 규칙 추가

## 삼삼 금지 규칙 상세
- **대상**: 흑돌만 적용 (백돌, AI는 제한 없음)
- **금지 조건**: 한 수를 뒀을 때 **양쪽이 모두 뚫린 삼(활삼)**이 동시에 2개 이상 만들어지는 경우
- **허용 조건**:
  - 한쪽이라도 막혀있는 삼(사삼)은 카운트 안 함
  - 4+3 조합은 허용 (삼이 1개만 생기는 경우)
  - 5목(승리)이 되는 수는 삼삼 체크 없이 승리 처리
- **UI 피드백**: 금지된 위치를 클릭/터치 시 "삼삼 금지" 토스트 메시지 표시

## 목표
HTML/CSS/JavaScript만으로 구현하는 웹 기반 오목 게임.
클릭(PC)과 터치(모바일) 모두 지원.

## 기술 스택
- 순수 HTML5 + CSS3 + Vanilla JS (외부 라이브러리 없음)
- Canvas API로 바둑판 및 돌 렌더링

## 게임 규칙
- 15x15 바둑판
- 흑돌 선공, 백돌 후공
- 가로/세로/대각선 방향으로 5개 연속 시 승리
- 삼삼 금지 규칙 적용 (위 상세 참고)

## 모드
1. **2인 대전** (기본): 같은 기기에서 번갈아 두기
2. **AI 대전**: 플레이어(흑) vs AI(백)

## UI/UX 요구사항
- 반응형 디자인: 모바일/PC 모두 전체 화면에 꽉 차게
- 바둑판: 전통적인 나무 느낌 (#DCB67A 계열)
- 흑돌/백돌 광택 효과
- 마지막으로 둔 돌에 빨간 점 표시
- 승리 시: 5개 연속된 돌 하이라이트 + 승리 오버레이
- 삼삼 금지 위치 클릭 시 토스트 메시지
- 현재 차례 표시, 게임 리셋 버튼
